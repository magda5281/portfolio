# Word Analytics

A simple React + Vite application for live text analysis, offering character and
word counts as well as remaining characters for Facebook and Instagram posts.

## Features

**Live Character Count:** Displays the total number of characters entered.

**Live Word Count:** Counts words by splitting on whitespace and filtering out
empty entries.

**Social Media Character Limits:** Shows characters remaining for:

Instagram: configurable via INSTAGRAM_MAX_CHARS

Facebook: configurable via FACEBOOK_MAX_CHARS

**Disallowed Input Warning:** Detects and warns when <script> tags or @ symbols
are entered.

**Responsive Design:** Uses pure CSS for styling and is fully responsive.

**Semantic HTML:** Clean, accessible markup.

**Reusable Components:** Modular structure for easy maintenance and extension.
