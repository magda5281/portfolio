  - Services: `UserService`, `ExpenseService`, `StorageService` manage state

  `toJSON()`â€”a clean boundary between app state and file format

  `ExpenseUI` (forms, select, lists, toasts)
- **Validation layer (optional):** pure functions for names, UUIDs, amounts,
  timestampsâ€”shared by services and import flow
- **Separation of concerns:** models (state/behavior) â†’ services (business
  rules) â†’ UI (render & events)
- **Testing:** Vitest + jsdom for models/services; DOM smoke tests for UI
