# MERN Thinkboard fullstack app

Key Features: 🧱 Full-Stack App Built with the MERN Stack (MongoDB, Express,
React, Node) ✨ Create, Update, and Delete Notes with Title & Description 🛠️
Fully Functional REST API ⚙️ Rate Limiting with Upstash Redis 🚀 Completely
Responsive UI 🌐 Explored HTTP Methods, Status Codes & NoSQL 📦 Deployed to
render.com
